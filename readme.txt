Ensure that you have python3 and pip installed
    `python -V`
    `pip -V`

cd to the project directory
    `cd location/you/downloaded`

to install the project requirements
	`sudo pip install -r requirements.txt`

to run the project
    `python main.py`

you can give the file name while running the program
    `python main.py path/to/xls/file`

you can also give the api key when running the program
    `python main.py path/to/xls/file apikey`

when putting the file name in the terminal,
open a finder window and drag and drop the file into the terminal to place the file name
